# Smart Dine - Restaurant Web App

Smart Dine is a React-based restaurant web application that allows users to browse menus and pre-order food with a clean and responsive interface.

## Features
- Browse food menu
- Pre-order food items
- Responsive UI design
- Reusable React components

## Tech Stack
- React.js
- JavaScript
- HTML5
- CSS3
- Bootstrap

## Installation
1. Clone the repository
2. Run npm install
3. Run npm start
